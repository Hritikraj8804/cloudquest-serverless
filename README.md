# CloudQuest ⚔️

CloudQuest is a fantasy-themed serverless quest management platform built on AWS.

## Architecture

Frontend (HTML/CSS/JS)
↓
API Gateway
↓
AWS Lambda
↓
DynamoDB

## Tech Stack

- AWS Lambda
- API Gateway
- DynamoDB
- Terraform
- GitHub Actions
- S3
- CloudFront
